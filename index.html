📗 Google Sheets ಇಂಪೋರ್ಟ್

1. URL ನಮೂದಿಸಿ
2. ಕಾಲಮ್ ಹೊಂದಿಸಿ
3. ಇಂಪೋರ್ಟ್
📌 ಹೇಗೆ ಮಾಡಬೇಕು:
1. Google Sheet ತೆರೆಯಿರಿ
2. File → Share → Publish to web ಆರಿಸಿ
3. Comma-separated values (.csv) ಆರಿಸಿ → Publish ಒತ್ತಿ
4. ಬಂದ URL ಕೆಳಗೆ ಹಾಕಿ
ಅಥವಾ: Sheet URL ನಿಂದ Sheet ID ತೆಗೆದು ಕೆಳಗೆ ಹಾಕಿ
⚠️ Sheet ಸಾರ್ವಜನಿಕ (Public) ಅಥವಾ "Anyone with link can view" ಆಗಿರಬೇಕು.
Sheet ಕಾಲಮ್‌ಗಳನ್ನು ವ್ಯವಸ್ಥೆಯ ಕ್ಷೇತ್ರಗಳಿಗೆ ಹೊಂದಿಸಿ. ಬೇಡದ ಕಾಲಮ್‌ಗಳನ್ನು — ತೆಗೆಯಿರಿ — ಆರಿಸಿ.
Sheet ಕಾಲಮ್ ಮಾದರಿ ಡೇಟಾ ಹೊಂದಿಸಿ →
⚠️ ದಾಖಲಾತಿ ಸಂಖ್ಯೆ ಒಂದೇ ಇದ್ದರೆ ಆ ವಿದ್ಯಾರ್ಥಿಯ ಮಾಹಿತಿ ಅಪ್‌ಡೇಟ್ ಆಗುತ್ತದೆ. ಹೊಸ ಸಂಖ್ಯೆ ಇದ್ದರೆ ಹೊಸ ವಿದ್ಯಾರ್ಥಿ ಸೇರಿಸಲಾಗುತ್ತದೆ.

ℹ️ ಪ್ರತಿ ಪ್ರಶ್ನೆಗೆ A (ಸಾಧಿಸಿದ್ದಾರೆ) ಅಥವಾ B (ಇನ್ನೂ ಸಾಧಿಸಬೇಕು) ಮಾತ್ರ ನಮೂದಿಸಿ

☁️ Firebase Sync ಸೆಟ್ಟಿಂಗ್

ಹೇಗೆ ಮಾಡುವುದು:
1. console.firebase.google.com ತೆರೆಯಿರಿ
2. New Project → Realtime Database → Create
3. Rules ನಲ್ಲಿ ".read":true, ".write":true ಹಾಕಿ
4. Project Settings → Your apps → Config ನಕಲಿಸಿ ಇಲ್ಲಿ ಹಾಕಿ
⚠️ Firebase ಸಂಪರ್ಕ ಆದ ನಂತರ "Local → Firebase Upload" ಮಾಡಿ ಮೊದಲ ಬಾರಿ data ಕಳಿಸಿ.
